## Instantly.io Setup Notes

### Variables to Configure
| Variable | Description | Example |
|----------|-------------|---------|
| `{{first_name}}` | Contact's first name | Sarah |
| `{{company_name}}` | Venue name | Pecan Springs Ranch |
| `{{email}}` | Contact email | sarah@venue.com |

### Recommended Settings
- **Daily sending limit:** Start with 30/day, increase gradually
- **Sending hours:** 8am-6pm CT (local Austin time)
- **Days:** Monday-Friday only
- **Reply detection:** Pause sequence on any reply
- **Bounce handling:** Remove on hard bounce

### A/B Testing Strategy
1. Test subject lines first (run for 100 contacts minimum)
2. Winner becomes default, test new challenger
3. Track: Open rate, reply rate, meeting rate

### Tagging Strategy
- `venue-wedding` - Wedding venues
- `venue-entertainment` - Boats, galleries, unique spaces
- `venue-barnranch` - Barns and ranches
- `venue-hillcountry` - Hill Country venues
- `responded` - Any response received
- `meeting-booked` - Call scheduled
- `partner-signed` - Partnership confirmed
